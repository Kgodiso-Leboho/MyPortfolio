## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Kgodiso-Leboho/MyPortfolio.git
cd MyPortfolio
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```
